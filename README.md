# Trabajo fin de máster

Este repositorio contiene la aplicación [WIPACE desarrollada para el TFG](https://github.com/sjm00010/TFG), con las mejoras de seguridad propuestas en el TFM.

## Despliegue

Para el despliegue de la aplicación se ha usado Vercel, ya que ofrece funcionalidades de seguridad interesantes, fáciles de configurar, y ademas la versión gratuita ofrece se adapta a las necesidades del TFM.
